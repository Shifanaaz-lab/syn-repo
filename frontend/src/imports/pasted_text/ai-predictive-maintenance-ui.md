Design a complete next-generation AI-powered predictive maintenance system interface that behaves and looks like a real-world industrial product already deployed in production.

This is NOT a simple dashboard. It must simulate a full system experience including authentication, system initialization, real-time data streaming, and an intelligent AI command center.

The design should feel like a high-end system similar to Tesla control systems, Jarvis (Iron Man), or a futuristic industrial AI monitoring platform.

---

## GLOBAL VISUAL STYLE

* Dark futuristic theme (deep black / navy base)
* Neon accents: electric cyan, magenta, amber
* Glassmorphism panels (blur + transparency)
* Soft glowing borders and light diffusion
* Holographic UI elements and HUD overlays
* Layered depth (avoid flat cards)
* Subtle gridlines, particles, or scanning effects
* Cinematic, premium, high-tech feel

---

## FULL SYSTEM FLOW (IMPORTANT)

1. LOGIN / AUTHENTICATION SCREEN

* System name: “NEXUS AI – Predictive Maintenance Core”
* Tagline: “Secure Industrial Intelligence System”
* Fields: Operator ID, Password
* Show:

  * System status (Operational)
  * AI Core (Active)
  * Last login info
* Add subtle animated background (grid/particles)
* Include “Authorized Access Only” messaging

2. SYSTEM INITIALIZATION / BOOT SCREEN

* Full-screen transition after login
* Show animated system logs:

  * Initializing AI Core…
  * Connecting to MongoDB Cluster…
  * Streaming telemetry channels…
  * Loading predictive model…
  * System Ready ✓
* Include progress indicators, scanning lines, terminal-style feel
* Must feel like a real system booting up

3. MAIN AI COMMAND CENTER (PRIMARY SCREEN)

* Layout should NOT be basic grid — use dynamic, layered structure

TOP BAR:

* System status (Operational / Warning / Critical)
* AI Core status (Active / Learning)
* Real-time clock
* Live indicator (blinking)
* Latency (e.g., 32ms)

LEFT SIDEBAR:

* Minimal icon-based navigation:

  * Dashboard
  * Engines
  * AI Insights
  * Alerts
  * Maintenance
  * Settings

---

## CORE MODULES (INSIDE DASHBOARD)

A. AI CORE CENTERPIECE (VISUAL FOCUS)

* Central glowing module representing AI system
* Circular / radial HUD design
* Show:

  * Model status (Active)
  * Confidence %
  * Learning/adaptation state
* Should feel like a “brain” of the system

B. LIVE TELEMETRY STREAM

* Real-time animated graphs (NOT static)
* Signals: Temperature, Pressure, Vibration
* Moving waveforms with glow
* Highlight anomalies with pulses

C. ENGINE NETWORK / DIGITAL TWIN

* Node-based visualization of engines
* Connections between nodes
* Critical engines glow/pulse red
* Clicking node expands detailed diagnostics

D. CRITICAL ENGINE FOCUS PANEL

* Detailed diagnostic view:

  * Failure probability %
  * Remaining Useful Life (RUL)
  * Sensor readings (°C, kPa, mm/s)
  * AI recommendation (natural language)
* Must look like a real diagnostic console

E. AI INSIGHTS STREAM (VERY IMPORTANT)

* Real-time scrolling feed (like logs)
* Messages such as:

  * “Anomaly detected in Engine E-037”
  * “Degradation rate exceeds baseline”
  * “Failure predicted within 17 cycles”
* Styled like an intelligent AI assistant output

F. FAILURE TIMELINE

* Visual progression:
  Stable → Degrading → Critical → Failure
* Show predicted future state

G. FLEET HEALTH & RUL DISTRIBUTION

* Overall health %
* Gradient bars for engines
* Ranked by risk level

H. DATA PIPELINE VISUALIZATION

* Show animated flow:
  Sensors → Streaming → MongoDB → AI Model → Prediction
* Represent with moving lines or flow diagram
